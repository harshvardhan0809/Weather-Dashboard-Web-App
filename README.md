The Weather Dashboard Web App allows users to search for real-time weather information of different cities using a public API. Users can view temperature, weather conditions, and other details, along with options to filter and sort results. The application aims to provide a clean and interactive interface for accessing weather data.

1. API Selection :-> 
OpenWeather API

2. Planned Features :->
    a. Search city weather
    b. Store multiple cities
    c. Filter by temperature / weather type
    d. Sort by temperature / name(a-z)
    e. Responsive UI 

3. TECH TO BE USED :-> 
    a. HTML
    b. CSS / Tailwind 
    c. JavaScript (to Fetch API)
